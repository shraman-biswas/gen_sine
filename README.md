Generate sine wave
==================

Generates and displays an animated sine wave.
Frequency, amplitude, display width, and animation time delay can be specified by command line arguments.

Usage: ./main frequency amplitude width delay  
&nbsp;&nbsp;
frequency: 0.1-10 (Hz)  
&nbsp;&nbsp;
amplitude: 0.0-1.0  
&nbsp;&nbsp;
width: 50-77  
&nbsp;&nbsp;
delay: 1-100 (s)
